ctrl+w [cerrar pestañas]

ctrl+f [buscar codigo dentro del archivo q estas usando]

Comandos git

git init [crea el repositorio]
git add . [agrega todos los archivos modificados]
git commit -m "mensaje" [manda los cambios al repositorio con un mensaje -m]
git diff [te indica el codigo q cambiaste]
git log [te muestra los commit]

antes de hacer el commit hacer un git add .
